## 🔎 서비스 소개

```
- 🧍‍♂️ 오래 앉아 있거나 특정 자세를 유지해야 하는데, 내 자세가 올바른지 확인하기 어려워요.

- 📹 실시간으로 자세를 분석하고 피드백을 받을 수 있는 시스템이 필요해요.

- 💻 장시간 컴퓨터를 사용하는 사람들에게 자세 교정이 필요하지만 스스로 교정하기 어려워요.

위와 같은 문제를 경험해보셨나요??
```
